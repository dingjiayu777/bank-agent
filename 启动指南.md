# 银行智能助手 - 启动指南

## 为什么显示"拒绝连接"？

"拒绝连接"通常表示 Streamlit 服务器没有成功启动。可能的原因：

1. **Python 未安装或不在 PATH 中**
2. **依赖包未安装**
3. **端口被占用**（8501 端口）
4. **启动命令错误**

## 解决步骤

### 方法 1: 使用启动脚本（推荐）

**Windows 用户：**
双击运行 `run.bat` 文件

**Mac/Linux 用户：**
```bash
chmod +x run.sh
./run.sh
```

### 方法 2: 手动启动

1. **检查 Python 安装**
   ```bash
   python --version
   # 或
   python3 --version
   ```
   如果没有输出，请先安装 Python 3.8+

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   # 或
   python -m pip install -r requirements.txt
   ```

3. **启动应用**
   ```bash
   streamlit run app.py
   # 或
   python -m streamlit run app.py
   ```

### 方法 3: 检查端口占用

如果 8501 端口被占用，可以指定其他端口：
```bash
streamlit run app.py --server.port 8502
```

## 常见错误及解决方法

### 错误 1: "找不到 streamlit 命令"
**解决方法：**
```bash
python -m pip install streamlit
python -m streamlit run app.py
```

### 错误 2: "ModuleNotFoundError"
**解决方法：**
```bash
pip install -r requirements.txt
```

### 错误 3: "Address already in use"
**解决方法：**
- 关闭占用 8501 端口的程序
- 或使用其他端口：`streamlit run app.py --server.port 8502`

### 错误 4: "拒绝连接"但应用已启动
**解决方法：**
- 检查防火墙设置
- 尝试访问 `http://127.0.0.1:8501` 而不是 `http://localhost:8501`
- 查看终端中的实际地址（Streamlit 会显示）

## 关于 OpenAI API Key

**重要：** 即使没有设置 OpenAI API Key，应用也应该能够正常启动并显示界面！

- 没有 API Key 时，你可以看到界面和账户信息
- 只有当你尝试与智能体对话时，才会提示需要设置 API Key
- 在侧边栏输入 API Key 后，点击"设置 API Key"按钮即可使用

## 验证应用是否成功启动

成功启动后，你应该看到：
```
You can now view your Streamlit app in your browser.

Local URL: http://localhost:8501
Network URL: http://192.168.x.x:8501
```

如果看到这些信息，说明应用已成功启动，请在浏览器中访问显示的地址。
